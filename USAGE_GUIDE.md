# Инструкция по использованию Hotel App

## Запуск приложения

1. Откройте проект в Android Studio
2. Дождитесь завершения синхронизации Gradle
3. Запустите приложение (Shift+F10 или кнопка Run)

## Навигация по приложению

### 1. Экран авторизации
- Введите логин и пароль (например, **1** / **1**)
- Нажмите "Войти"
- Для создания нового пользователя нажмите "Регистрация"

### 2. Главное меню
После успешной авторизации вы увидите:
- Приветствие "Здравствуйте!"
- Ваше ФИО
- Три кнопки:
  - **"Поиск номеров по фильтрам"** - поиск доступных номеров на определенные даты
  - **"Просмотр всех номеров"** - просмотр всех номеров с возможностью фильтрации
  - **"Выйти из учетной записи"** - выход и возврат к экрану авторизации

### 3. Поиск по фильтрам
Нажав "Поиск номеров по фильтрам", вы попадете на экран с полями:
- **Дата заезда** - нажмите на поле, выберите дату в календаре
- **Дата выезда** - нажмите на поле, выберите дату в календаре
- **Количество гостей** - введите число
- **Кнопка "Найти"** - запустит поиск
- **Кнопка "Назад"** - вернет в главное меню

После нажатия "Найти" откроется список номеров, соответствующих вашим критериям.

### 4. Просмотр всех номеров
При нажатии "Просмотр всех номеров" откроется список со всеми номерами отеля.

Вверху экрана три кнопки для фильтрации:
- **"Все"** - показать все номера
- **"Доступные"** - показать только свободные номера
- **"Недоступные"** - показать только занятые номера

### 5. Детали номера
При нажатии на любой номер в списке откроется экран с детальной информацией:
- Изображение номера (пока заглушка - серый прямоугольник)
- Номер комнаты (например, "Номер 101")
- Тип номера (Эконом, Стандарт, Делюкс, Люкс, Семейный, Президентский)
- Вместимость (количество гостей)
- Цена за ночь
- Описание номера
- Удобства (Wi-Fi, Телевизор, Кондиционер, Минибар, Балкон)
- Статус: Доступен / Занят

Кнопка "Назад" вернет вас к списку номеров.

## Добавление фотографий номеров

В текущей версии используются placeholder изображения (серые прямоугольники).

Чтобы добавить реальные фотографии:

1. Поместите файлы изображений в папку `app/src/main/res/drawable/`
   - Названия файлов должны быть в формате: `room_101.jpg`, `room_102.jpg` и т.д.
   - Используйте только строчные буквы и цифры (без пробелов)

2. Откройте файл `DatabaseHelper.kt`

3. Найдите метод `insertTestData()` и замените строки типа:
   ```kotlin
   put(COL_ROOM_IMAGE, "placeholder")
   ```
   на:
   ```kotlin
   put(COL_ROOM_IMAGE, "room_101")  // без расширения .jpg
   ```

4. Обновите код в `RoomAdapter.kt` и `RoomDetailsActivity.kt`:
   
   **RoomAdapter.kt**, метод `bind()`:
   ```kotlin
   // Вместо:
   ivRoomImage.setImageResource(R.drawable.placeholder_room)
   
   // Используйте:
   val imageResId = itemView.context.resources.getIdentifier(
       room.imagePath, "drawable", itemView.context.packageName
   )
   if (imageResId != 0) {
       ivRoomImage.setImageResource(imageResId)
   } else {
       ivRoomImage.setImageResource(R.drawable.placeholder_room)
   }
   ```

   **RoomDetailsActivity.kt**, метод `displayRoomDetails()`:
   ```kotlin
   // Вместо:
   ivRoomDetailImage.setImageResource(R.drawable.placeholder_room)
   
   // Используйте:
   val imageResId = resources.getIdentifier(
       room.imagePath, "drawable", packageName
   )
   if (imageResId != 0) {
       ivRoomDetailImage.setImageResource(imageResId)
   } else {
       ivRoomDetailImage.setImageResource(R.drawable.placeholder_room)
   }
   ```

5. Удалите базу данных приложения (чтобы пересоздать с новыми данными):
   - В Android Studio: Tools → Device File Explorer
   - Перейдите в `/data/data/com.example.kvalik_hotel/databases/`
   - Удалите файл `hotel.db`
   - Или просто переустановите приложение

## Техническая информация

- **Язык программирования**: Kotlin
- **Минимальная версия Android**: API 24 (Android 7.0)
- **Целевая версия Android**: API 35
- **База данных**: SQLite
- **UI**: Material Design Components
- **Архитектура**: Activity-based с SQLiteOpenHelper
